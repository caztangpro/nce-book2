🛣️ 1. Way 家族 —— 四个 way 短语各不同

on the way / in the way / in a way / by the way L14
🛣️ on the way
途中，在路上
On the way, a man waved to me.
🚧 in the way
挡路，妨碍
Don't stand in the way.
🤔 in a way
从某种意义上说
In a way, you're right.
💬 by the way
顺便说一下
By the way, I saw him yesterday.
🗝 记忆钥匙：看介词本意
on（接触表面）= 在路上走 → on the way
in（在里面）= 堵在路里面 → in the way（挡路了）
in a（在某范围内）= 在某种意义上 → in a way
by（顺便路过）= 顺便提及 → by the way

❓ 2. "问与答"辨析

ask / ask for / reply / answer L14
短语用法例子
ask sb sth问某人某事（及物）I asked him his name.
ask for sth请求得到某物He asked for a lift.
ask for sb找某人（询问）Someone asked for you.
reply (to sb)回答（不及物，+to）He replied in the same language.
answer sb/sth回答（及物，直接+宾语）He answered my question.
🗝 记忆钥匙：ask + for = 想要得到（for = 为了）
reply = 回答（较正式，不及物，不能说 reply me ❌）
answer = 回答（直接接宾语，answer the question ✅）

➗ 3. "除了"三兄弟

apart from / except / besides L14
短语含义包含/排除例子
apart from 除了……（双重含义） 看语境，可包可排 Apart from a few words, I don't know French.
（排除——除了几个词以外都不会）
except 除了……（排除在外） ❌ 排除 Everyone except me knew French.
（我除外，别人都会）
besides 除了……（还有） ✅ 包含 Besides French, he speaks English.
（除了法语，还会英语）
🗝 记忆：except = 排除（ex- = 向外，把……排除出去）
besides = 包含（side = 旁边，在……旁边还有）
apart from = 中立派（两个意思都可能）

🎯 4. "两者/多者"否定与肯定

neither of / either of / none of / both of L14
短语范围含义例子
neither of两者（两者）都不Neither of us spoke.
either of两者（两者中）任一Either of us can help you.
both of两者（两者）都Both of them speak French.
none of三者或以上（所有）都不None of my friends live here.
all of三者或以上（所有）都All of them were late.
🗝 记忆：n- 开头 = 否定（neither/none）
e- 开头 = 二选一（either）
b- 开头 = 两者都（both）
既分数量又分肯定/否定

⛵ 5. "出发"的各种说法

set out / set off / set sail L12
短语含义用法例子
set out 出发 最通用，不限交通方式 Captain Alison set out at eight o'clock.
set off 出发 与 set out 基本同义 We set off early in the morning.
set sail 起航 特指船只出发 They set sail for America.
🗝 记忆：sail = 航行 → 只用于船。out / off 通用。

👥 6. "参加"的三种用法

take part in / join / attend L12
短语含义搭配例子
take part in 参加（活动） 赛事、竞赛、游戏 He will take part in an important race.
join 加入（组织/团体） 军队、俱乐部、公司 He joined the army.
attend 出席（场合） 会议、课程、仪式 I attended the meeting.
🗝 记忆层级：take part in = 参与活动 / join = 加入组织 / attend = 出席场合

⏰ 7. "通常"的不同表达

as usual / usually / at present L13 · L14
表达词性含义例子
as usual 副词短语 "像往常一样"（特定情况） As usual, the police had a difficult time.
usually 频率副词 "通常"（一般情况下，习惯） I usually get up at 7.
at present 时间副词 "目前"（= now，稍正式） At present, they are visiting all parts.
🗝 记忆：as usual = "像往常一样"（当天/当时的情况）
usually = "通常"（长期的习惯）

🔒 8. keep 的常见搭配

keep order / keep calm / keep quiet / keep doing L13
搭配含义例子
keep order维持秩序The police will try to keep order.
keep calm保持冷静Keep calm and carry on.
keep quiet / silent保持安静/沉默Please keep quiet.
keep + doing持续做某事He kept talking for an hour.
keep + adj保持某种状态Keep safe / Keep warm.
