与现在完成时被动相同模式

🧩 规律总结：被动语态的统一公式

所有被动语态都遵循同一个模式：

主动：时态标记
will / have / is / was …
+
核心操作
be
+
过去分词
done

被动 = 把主动的动词换成 "be + 过去分词"，be 继承主动语态的时态形式。

  • 主动用 do / does → 被动 be = am / is / are
  • 主动用 did → 被动 be = was / were
  • 主动用 will do → 被动 be 不变 = be（因为是原形）
  • 主动用 have done → 被动 be = been
  • 主动用 had done → 被动 be = been（had + been）
  • 主动用 is doing → 被动 = is being done（此时 being 是进行时的标志）
  • 主动用 was doing → 被动 = was being done
  • 主动用 will be doing → 被动 = will be being done
💡 核心记忆法：
把 do 换成 be + done。
be 保留原来的时态 → 你就得到了被动语态。

📈 各时态被动语态的使用频率

使用频率时态说明
🔥 频繁 一般现在时被动 / 一般过去时被动 科技文章、新闻中最常见：It is said that… / He was born in…
✅ 常见 现在完成时被动 / 一般将来时被动 / 现在进行时被动 正式写作和口语中合理使用
⚠️ 较少 过去进行时被动 / 过去完成时被动 特定语境才出现，但不罕见
❌ 极少 将来进行时被动 语法存在，实际几乎不用（见下文）

🎯 专题：将来进行时被动 (will be being done)

语法正确，但实际交流中几乎不用。

为什么？因为 三个辅助动词堆在一起（will + be + being），听起来非常笨重：

The concert will be being held at 8pm.

英语母语者的改写策略：

改写方式例子
改用一般将来被动 The concert will be held at 8pm.（够了，不需要进行）
用 under + 名词 The stadium will be under construction next year.
用 in progress The work will be in progress at that time.
用从句改写 At that time, construction of the stadium will be ongoing.

🧠 容易混淆的两组

being done vs been done

含义"正在被……""已经被……"

💡 记忆：
being = be + ing（进行中）→ "正在被"
been = be + en（完成标志）→ "已经被"
