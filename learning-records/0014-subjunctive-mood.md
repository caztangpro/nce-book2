# 虚拟语气 (Subjunctive Mood)

## 核心概念
虚拟语气表达"不是事实"的内容——假设、愿望、建议、怀疑。

## 四类条件句核心公式
- 🟢 零类（真理）：If + 一般现在时, 一般现在时
- 🔵 第一类（很可能）：If + 一般现在时, will + 原形
- 🟠 第二类（反现在）：If + 一般过去时 (were), would + 原形
- 🔴 第三类（悔过去）：If + had done, would have + 过去分词
- 🟣 混合（从句过去→主句现在）：If + had done, would + 原形

## 其他虚拟语气场景
- wish + 过去时（现在）/ had done（过去）/ would（将来）
- as if + 虚拟时态
- suggest/insist/recommend + that + (should) + 动词原形
- It's time + 过去时 / would rather + 过去时

## 关键易错点
- 第二类 if 从句中 be 动词永远用 were（不用 was）
- if 从句里不能出现 would/will
- suggest 从句中动词用原形（不加 s）

**Why:** 虚拟语气是英语表达假设、后悔、委婉请求的核心语法，也是各类英语考试的高频考点。
**How to apply:** 判断要表达的内容是不是"事实"——不是事实就用虚拟语气；然后根据时间判断用第二类还是第三类。
