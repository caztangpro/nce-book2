# 0003 · Lesson 3: Please Send Me a Card

## 日期
2026-06-15

## 学习内容
- **课文**：去意大利度假被明信片"困住"的幽默故事（买了37张但一张没写）
- **语法**：一般过去时（叙事的核心时态）
  - 规则动词过去式：-ed (visited, passed, decided)
  - 不规则动词：go→went, teach→taught, lend→lent, read→read, think→thought, make→made, buy→bought, spend→spent, write→wrote, send→sent, spoil→spoilt, get up→got up
  - 否定句：did not + 动词原形
  - 疑问句：Did + 主语 + 动词原形?
- **词汇**：send, postcard, spoil, museum, public, friendly, waiter, lend, decision, whole, single
- **特殊难点**：spend time (on) / spend time doing；a few vs few；send + 双宾语
