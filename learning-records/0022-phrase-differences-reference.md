# 易混短语辨析速查参考文档

## 背景

Lesson 14 课后学生追问 "in the way 和 on the way 的区别"，由此引申出将之前所有课程中出现的易混短语集中到一个统一参考页中的需求。

## 创建内容

`reference/phrase-differences.html` 包含 9 个辨析专题：

1. **Way 家族** (L14)：on the way / in the way / in a way / by the way
2. **问与答辨析** (L14)：ask / ask for / reply / answer
3. **除了三兄弟** (L14)：apart from / except / besides
4. **两者/多者** (L14)：neither of / either of / none of / both of
5. **出发的各种说法** (L12)：set out / set off / set sail
6. **参加的三种用法** (L12)：take part in / join / attend
7. **通常的不同表达** (L13)：as usual / usually / at present
8. **keep 搭配** (L13)：keep order / keep calm / keep quiet
9. **其他固定搭配** (L4-L14)：be proud of, plenty of, by + 交通工具等
