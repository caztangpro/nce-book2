# 0008 · Lesson 6: Percy Buttons

## 日期
2026-06-15

## 学习内容
- **课文**：倒立唱歌换饭吃的乞丐 Percy Buttons 的故事
- **语法**：冠词 a/an（第一次提到）vs the（再次提到/已知）
  - 课文示范：a beggar → the beggar / a meal → the food
  - an 用在元音音素前：an engineer, an hour
  - 不加冠词：不可数名词泛指（water, beer 作泛指）
- **词汇**：beggar, knock, meal, beer, cheese, pocket, neighbour
- **特殊难点**：
  - some（肯定句/请求）vs any（否定句/疑问句）
  - knock at（敲门）vs knock down（撞倒）
  - call at（拜访地方）vs call on（拜访人）
  - in return for（作为回报）
