# Lesson 12: Goodbye and Good Luck

## 主要语法点：一般将来时

### will vs be going to
- **will** + 动词原形 = 临时决定 / 个人预测 / 陈述将来事实
- **be going to** + 动词原形 = 早有安排 / 有迹象即将发生
- 课文主要使用 will（叙述计划中的事件）

### will 的结构
- 肯定：主语 + will + 动词原形
- 否定：主语 + will not (won't) + 动词原形
- 疑问：Will + 主语 + 动词原形?

### 表达将来的其他方式
- 一般现在时表将来：按时刻表确定（set out at eight o'clock）

### 其他知识点
- set out / set off / set sail 出发的不同用法
- plenty of + 可数/不可数名词（大量）
- be proud of（以……为自豪）
- take part in（参加活动）
- for + 时间段（离开……时间）

### 课文
为查尔斯·艾利森船长送行，他将驾驶小船 Topsail 参加横渡大西洋比赛。

**Why:** 一般将来时是日常英语最常用的时态之一，正确区分 will 和 be going to 能提升表达的自然度。
**How to apply:** "临时决定"用 will，"早有安排"用 be going to——想想自己说话时是刚刚决定还是计划好的。
