# would 的用法

## 问题来源
Lesson 9 课文中 "It would strike twelve in twenty minutes' time." 中的 would 不理解。

## 解答：过去将来时 (Future in the Past)

在过去的叙事语境中，表达"站在过去看将要发生的事情"要用 would（而不是 will）。

- will → 直接说未来：*It will strike twelve.*（它将会敲响十二点）
- would → 过去视角看未来：*It would strike twelve.*（当时看，它将要敲响十二点）

## 补充：would 的六大用法
1. **过去将来时** — 课文用法，过去视角的未来
2. **虚拟条件句** — if I were you, I would...
3. **礼貌请求** — Would you like...? / I would like...
4. **过去习惯** — When I was young, I would...（= used to，但只用于动作）
5. **拒绝/不配合** — The car wouldn't start.（= refused to）
6. **委婉推测** — That would be my mother calling.

**Why:** would 是英语中最常用的情态动词之一，用法多样，是理解复杂句子的关键。
**How to apply:** 看到 would 时先判断它是哪种用法——最常见的是"礼貌"和"虚拟语气"，但课文中这个"过去将来"很容易被忽略。
