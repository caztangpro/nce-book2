# 0004 · 数量限定词：a few / a little / a bit / a few of

## 日期
2026-06-15

## 来源
Lesson 3 课后讨论时主动提问

## 知识点
- a few → 可数名词复数（泛称"几个"）
- a little → 不可数名词（"一点儿"），也可修饰形容词
- a bit (of) → 口语化的 a little，也可修饰形容词
- a few / little / bit of + 限定词 → 特指"…中的几个/一点儿"
- few / little（不加 a）→ 表否定含义"几乎没有"
