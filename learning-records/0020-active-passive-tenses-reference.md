# 主动语态与被动语态时态对照参考

## 背景

Lesson 13 课后学生追问"将来进行时被动怎么说"，由此引申出对所有时态主动/被动对比的系统梳理。

## 核心内容

创建了 `reference/active-passive-tenses.html` 参考文档，包含：

### 8 大时态主动/被动对照表

| # | 时态 | 主动公式 | 被动公式 |
|---|------|---------|---------|
| 1 | 一般现在时 | do/does | am/is/are + done |
| 2 | 一般过去时 | did | was/were + done |
| 3 | 一般将来时 | will do | will be + done |
| 4 | 现在进行时 | is/am/are + doing | is/am/are + being + done |
| 5 | 过去进行时 | was/were + doing | was/were + being + done |
| 6 | **将来进行时** | will be + doing | will be + being + done ⚠️ |
| 7 | 现在完成时 | have/has + done | have/has + been + done |
| 8 | 过去完成时 | had + done | had + been + done |

### 关键见解

1. **统一规律**：所有被动 = 把主动动词换成 **be + done**，be 继承主动语态的时态形式
2. **将来进行时被动 (will be being done)**：语法正确，但三个辅助动词堆叠极为笨重，实际交流中几乎不用。母语者会改写为一般将来被动或用 under + 名词等结构替代
3. **being vs been** 辨析：being = be+ing（进行中→"正在被"），been = be+en（完成态→"已经被"）
